# Raghav Pant — Portfolio

A personal portfolio built from scratch — no templates, no frameworks, just HTML, CSS, and vanilla JavaScript. Designed with a retro terminal aesthetic because blending in is boring.

🌐 **Live site:** https://raghavpant.vercel.app/

---

## What's inside

- Dark retro theme with a CRT scanline texture
- Typewriter effect cycling through phrases
- Project cards for EcoTwin, CodeForces Analyzer, and WeatherApp
- Smooth hover animations throughout
- Fully responsive — works on mobile too
- Auto-deploys to Vercel on every push

---

## Built with

| Layer | Tech |
|-------|------|
| Structure | HTML5 (semantic) |
| Styling | CSS3 (variables, flexbox, grid, animations) |
| Behaviour | Vanilla JavaScript |
| Fonts | Space Grotesk + JetBrains Mono |
| Hosting | Vercel |

---

## Run locally

No setup needed. Just clone and open:

```bash
git clone https://github.com/raghav1057/portfolio.git
cd portfolio
```

Then open `index.html` in any browser.
